# Landing Page - Dentista

Landing page profissional desenvolvida para apresentação de serviços odontológicos.

## Tecnologias Utilizadas
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Como executar localmente
1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador ou utilize um servidor local (como o Live Server do VS Code).

## Estrutura do Projeto
- `index.html`: Estrutura da página
- `style.css`: Estilização e design responsivo
- `script.js`: Lógica de interatividade e animações
- `assets/`: Imagens e ícones
